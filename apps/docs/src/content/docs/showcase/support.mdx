The React Native Reusables App (“the App”) is an interactive developer tool that enables users to explore and implement UI components and blocks provided by the React Native Reusables project.

### 1. How to Get Support
If you need help with the App or React Native Reusables, you can:
- Browse the documentation available on this website.  
- Open a discussion or report an issue on the [GitHub repository](https://github.com/mrzachnugent/react-native-reusables).  
- Reach out directly by email at: **mrzachnugent@gmail.com**

### 2. Response Time
Support inquiries will be reviewed as time permits. While there is no guaranteed response time, every effort will be made to address questions and issues promptly.

### 3. Updates to Support Information
Support details may be updated in the future. When changes occur, they will be reflected on this page.
